Installation



Obs.: This mod uses OpenIV Package Installer, see how to install this kind of mod here

Install order:



  1. Godzilla by JulioNIB.oiv - This will install the script, choose Game folder
  2. Suits config.oiv - This will install suit .ini files for some ped models, choose Game folder (links for the ped models in "Godzilla ped models.txt")
  3. Godzilla props.oiv - This will install some custom props used by the script, choose Mods folder (Atomic Breath beam, etc.)
  4. Download and install ped models (Check ped models text file)
  5. (Optional) Install game crash fix for added peds

Case the added props doesnt work for you (Atomic Breath effect not appearing), install the "Godzilla props (replace).oiv" package (choose Mods folder) and also download and install this package


Use the Suit config menu to adapt the script to other ped models, case you need :)


Issues:


-Avoid water

-Some things on the path may stop you from moving, use jump or turn camera and walk away

Don't forget to install the NIBShDotNet plugin (that includes the NIBMods Menu) and the ScriptHookV

Check the mod Readme.txt file for important info about requirements



Hotkeys

Menus:

    Ctrl + N - Show mods

    Up, Down, Left or Right (also numpads 2, 4, 6 and 8) - Navigate

    Enter or Numpad5 - Select item

    Numpad0 or Backspace - Previous menu/Close menu


In controller:

    Hold DPad Left and press X to show the NIBMods

    Up, Down, Left or Right - Navigate

    A (Sprint) - Select item

    B - Previous menu/Close menu



Gameplay


    Press Ctrl+Zero (the zero close to P key) to see the NIBMods spawn


    R - Special Attack, hold to charge/keep)

    E - Hold to select special attack

    Left Mouse button - Melee attacks (hold to charge the melee attack in Mechagodzilla)

    Space - Jump or flight (Depends on suit config.)

    Q - Grab target

    Q with grabbed entity - Release

    Left Mouse button with grabbed entity - Throw

    Press 1, 2, 3... to perform the default special attacks, you can change those hotkeys/attacks in the mod menu (Set hotkeys for special attacks)


    Controller (Under development)

    Right trigger - Special attack

    B - Melee attacks

    X - Toggle flight or Jump

    Y - Grab target (hold aim to enter vehicle)

    DPad Right - Hold to show special attack menu



You can edit the hotkeys using the mod Controls menu in the mod Options

Explore the mod Options menu, there is few customization options there

Explore the mod Suit config. menu, lot of customizations there.
